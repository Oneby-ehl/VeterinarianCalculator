# 🦉 Calculadora de dosis · Fauna silvestre

Aplicación web progresiva (PWA) para cálculo rápido de dosis y pautas de hidratación.

Diseñada para uso práctico en campo o clínica, con funcionamiento offline.

## 🚀 Acceso

https://oneby-ehl.github.io/VeterinarianCalculator/

## ✨ Funcionalidades

* Cálculo de dosis (mg/kg → ml)
* Cálculo de hidratación (SC y VO)
* Funcionamiento offline tras la primera carga
* Instalable como app (móvil y escritorio)
* Interfaz rápida y sencilla

## 📱 Instalación

1. Abre la app en el navegador
2. Selecciona "Añadir a pantalla de inicio" o "Instalar"
3. Podrás usarla sin conexión

## 📴 Uso offline

Una vez cargada o instalada, la aplicación funciona sin conexión a internet.

## ⚠️ Aviso importante

Herramienta de apoyo al cálculo.

* No sustituye el criterio veterinario ni protocolos clínicos
* Verificar siempre dosis, concentraciones y unidades
* El uso es responsabilidad exclusiva del usuario
